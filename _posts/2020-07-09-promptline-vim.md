## Tl;dr
  1. Install <https://github.com/edkolev/promptline.vim> script in `vim`
  2. Start `vim`, and run `:PromptlineSnapshot ~/.promptline.sh airline`
  3. In `~/.bashrc` append `source ~/.promptline.sh`
